Checklist thao tác thủ công

  1. Quản lý tài khoản: Thêm → Sửa → Xóa → Nhập/Xuất CSV.
  2. Quản lý proxy: Thêm → Sửa → Xóa → Gán proxy cho tài khoản đã chọn.
  3. Quản lý video: Chọn thư mục → Quét video → Chọn video.
  4. Đăng TikTok: Chọn Edge profile → chọn video/caption → Bắt đầu → kiểm tra video/caption đã được chèn chưa.
  5. Cloud Sync: Test Supabase → Push → Pull → Export Google Sheets.
  6. AI: mở ChatGPT/Gemini Web bằng đúng Edge profile, test API nếu có key.

Sub Douyin/Việt

Chọn video tiếng Trung từ máy, app sẽ tách audio, nhận diện lời thoại, dịch sang tiếng Việt và xuất SRT/TXT. Có thể xuất thêm MP4 gắn phụ đề Việt nếu máy có FFmpeg. Nếu Gemini 503/quá tải, app sẽ tự retry và đổi model fallback. Nếu API bị giới hạn, có thể dùng nút web fallback để mở ChatGPT/Gemini web trong app và copy prompt xử lý thủ công.

Thiết lập dịch

Chưa chọn, app sẽ dùng thư mục Videos.

Định dạng khung video đầu ra

App sẽ scale + pad để giữ đúng tỉ lệ gốc, tránh méo hình. Phù hợp cho Facebook, TikTok, YouTube và các nền tảng mạng xã hội.

Style phụ đề tiếng Việt

Khuyến nghị: chữ đen + nền vàng/trắng để dễ đọc. Font Arial/Segoe UI/Tahoma hỗ trợ tiếng Việt tốt trên Windows.

Chưa chọn FFmpeg.

FFmpeg cần để tách audio và gắn phụ đề. Có thể chọn trực tiếp file ffmpeg.exe, không cần thêm PATH.

Voice Việt — TTS OhFree

Tích hợp web TTS OhFree vào app để tạo voice tiếng Việt. Do chưa có API chính thức công khai, app mở web trong cửa sổ riêng và hỗ trợ copy nhanh nội dung tiếng Việt để dán vào web.

Nội dung cần đọc

Cách dùng: Copy nội dung → Mở TTS OhFree → chọn voice tiếng Việt → dán nội dung → generate/download audio.

Ghi chú tích hợp

Hiện app không gọi API ẩn của TTS OhFree để tránh hỏng khi web đổi request hoặc vi phạm điều khoản. Nếu Jordan có tài liệu API/chìa khóa chính thức của dịch vụ này, có thể nâng cấp thành tạo voice tự động.
